About This App

ToDo List App is a kind of app that generally
used to maintain our day-to-day tasks or list
everything that we have to do. It is helpful
in planning our daily schedules.

In this version of the ToDo List App, the user will be getting four options:

  • Create (add) a new task or adding a new ToDo in the ToDo List App.
  • See all the tasks or View all the ToDos that were added to the app.
  • Changing the already created task by clicking "change" button.
  • Delete the created ToDo from ToDos list.
